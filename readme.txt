Native Android App Builder for wordpress and woocommerce.

== Description ==

Native Android App Builder for wordpress and woocommerce. Build your native mobile app inside wordpress dashboard. No need to hire a mobile app developer. All the data from your wordpress website. Use custom post types and Advanced Custom Fields plugin if you need more than regular wordpress post data. The build process also runs from your wordpress dashboard and will generate apk and aab files to upload on the google play store. All of that is available for free.

It is like guttenberg editor, elementor or other wordpress builder but for generating a native mobile app. Very flexible editor with keyboard shortcut support. For example, to move the position of a button or image, you can do it by copy and paste using keyboard shortcuts.

== Frequently Asked Questions ==

= Can I generate android apk or aab file using this free version? =

Yes, You can. The free version is a completed version from pages layout to build your android apk and aab files. Are all available for free.

= Is this plugin need third-party services? =

Yes, it is. The build process needs a Github account and an Expo account. Both are available for free. The Github API to store the generated code. As for the Expo API, is to build the apk and aab files and for push notifications. 

= Can We upload the generated apk or aab file to the google play store? =

Yes, You can. The apk and aab files are generated on the cloud using expo services. The generated apk and aab file will be available to download on your expo account. 

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload `mobile-app-editor.zip` to the `/wp-content/plugins/` directory
2. Extract the zip file
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Use the Mobile App Editor > screen to create apps and pages
